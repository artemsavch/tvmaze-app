## Project structure

```
tvmaze-app/
├── backend/     # Node.js + Express server with TypeScript
├── frontend/    # Next.js app with TypeScript and Tailwind
├── .env-example # .env example
├── Makefile     # Recipes for quick start
└── README.md
```

---

## Get started

### 1. Install dependensies

```bash
make install
```

### 2. Start the project

```bash
make dev
```

## Testing

### Backend

```bash
make test-backend
```
